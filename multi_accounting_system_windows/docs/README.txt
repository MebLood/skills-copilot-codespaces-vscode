# بسته مولتی‌اکانتینگ هوشمند برای ویندوز

## نحوه استفاده:
1. `start.bat` را اجرا کنید تا پیش‌نیازها نصب شوند.
2. پس از اتمام نصب، سیستم را ری‌استارت کنید.
3. `run.bat` را اجرا کنید تا سیستم مولتی‌اکانتینگ راه‌اندازی شود.

این بسته به صورت خودکار همه مراحل را انجام می‌دهد و از WSL، Docker، Minikube و مرورگر Tor برای مدیریت حساب‌های جداگانه استفاده می‌کند.
