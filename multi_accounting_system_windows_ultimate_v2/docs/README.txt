# 📌 بسته مولتی‌اکانتینگ هوشمند برای ویندوز

## ✅ نحوه استفاده:
1. `start.bat` را اجرا کنید تا پیش‌نیازها نصب شوند.
2. پس از اتمام نصب، **سیستم را ری‌استارت کنید.**
3. `run.bat` را اجرا کنید تا سیستم مولتی‌اکانتینگ راه‌اندازی شود.

## 🚀 در صورت وجود مشکل:
🔹 **اگر Minikube یا Docker اجرا نشدند، ابتدا `reset.bat` را اجرا کنید، سپس `run.bat` را مجدداً اجرا کنید.**  
🔹 **اگر همچنان مشکلی وجود داشت، سیستم را ری‌استارت کنید و دوباره `run.bat` را اجرا کنید.**

🚨 **این بسته کاملاً خودکار است و تمامی پیش‌نیازها را بدون نیاز به کاربر نصب و اجرا می‌کند.** 🚀
